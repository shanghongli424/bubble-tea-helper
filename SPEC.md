# 奶茶调配助手 - Bubble Tea Master

## Concept & Vision

一个温暖、实用的奶茶店培训工具。新员工打开手机，点击对应饮品，跟着语音+文字+图文的步骤一步一步做，**无需背配方**。界面像一张干净的配方卡，有手作感但不土，让人想认真跟着做。

## Design Language

### Aesthetic Direction
温暖的奶茶色调，带点日式杂货铺的质朴感。不是网红店那种过度美化的风格，而是**踏实、清晰、让人信任**的工具感。

### Typography
- 标题: **Noto Sans SC** 700, 备用 system-ui
- 正文: **Noto Sans SC** 400/500
- 步骤数字: **Playfair Display** 700 (有种优雅配方卡的感觉)

### Spatial System
- 基础单位: 8px
- 卡片圆角: 16px
- 按钮圆角: 12px (大触控区域)
- 间距: 16px / 24px / 32px

### Motion Philosophy
- 步骤切换: slide + fade, 300ms ease-out
- 按钮按下: scale(0.97), 100ms
- 语音播放时: 步骤卡片有轻微 pulse 效果
- 进度条: 平滑过渡 400ms

### Visual Assets
- 步骤图标: Emoji (🧋 🍵 🥛 🧊 🍯 🫙 🥤)
- 饮品图标: Emoji (🧋)
- 装饰: 淡淡的波浪线分隔符

## Features & Interactions

### 核心功能

1. **饮品选择**
   - 点击饮品卡片 → 进入该饮品的步骤页
   - 支持搜索过滤（实时匹配饮品名）

2. **步骤导航**
   - 大按钮"下一步"推进到下一步
   - 小按钮"上一步"返回上一步
   - 进度点显示当前位置
   - 步骤完成后自动显示完成页

3. **语音朗读 (TTS)**
   - 点击"朗读步骤"按钮或步骤卡片 → 朗读当前步骤文字
   - 使用 Web Speech API (中文语音)
   - 朗读时按钮变为"🔊 朗读中..."，有 pulse 动画
   - 可重复点击重复朗读

4. **步骤卡片**
   - 显示：步骤号、图标、标题、详细文字说明、示意图（可选）
   - 当前步骤高亮，已完成步骤变灰

### 交互细节

- **按钮按下**: scale(0.97) + 背景变深
- **步骤切换**: 向左滑入 (translateX + opacity)
- **搜索框**: 聚焦时边框变焦糖色
- **卡片 hover/focus**: 轻微上浮 + 阴影加深
- **进度点**: 当前步放大 + 高亮，完成步填充

### 边界情况
- 第一步时隐藏"上一步"按钮
- 最后一步时"下一步"变为"完成！"
- 搜索无结果时显示"未找到该饮品"
- 语音不支持时隐藏朗读按钮

## Component Inventory

### 1. Header (标题栏)
- 左侧: 返回按钮 (步骤页) / Logo (首页)
- 中间: 页面标题
- 状态: 默认

### 2. SearchBar (搜索框)
- placeholder: "搜索饮品..."
- 状态: 默认、聚焦、有内容

### 3. DrinkCard (饮品卡片)
- 显示: 图标 + 饮品名 + 简介
- 状态: 默认、hover/press、focus

### 4. ProgressDots (进度点)
- 圆点序列，当前步高亮放大
- 状态: 待完成(空心)、当前(放大+实心)、已完成(实心)

### 5. StepCard (步骤卡片)
- 显示: 步骤图标(大emoji) + 步骤号 + 标题 + 描述 + 插图
- 状态: 朗读中(pulse)、默认

### 6. PrimaryButton (主按钮)
- "下一步" / "完成！"
- 状态: 默认、按下、禁用

### 7. SecondaryButton (次要按钮)
- "上一步" / "朗读步骤"
- 状态: 默认、按下、禁用、朗读中

### 8. CompletePage (完成页)
- 庆祝动画 + 按钮组

## Technical Approach

### 架构
- **单 HTML 文件**: 无需构建工具，复制部署
- **纯前端**: Vanilla JS + CSS，无框架依赖
- **数据内嵌**: 饮品配方以 JSON 形式写在 JS 中

### 数据模型
```javascript
const drinks = [
  {
    id: "pearl-milk-tea",
    name: "珍珠奶茶",
    category: "经典奶茶",
    intro: "最经典的国民饮品",
    steps: [
      {
        order: 1,
        icon: "🍵",
        title: "加入茶汤",
        description: "将 300ml 茉莉绿茶倒入 700ml 雪克杯中",
        image: null // 可选，base64 或 URL
      },
      // ...
    ]
  },
  // ...
]
```

### 语音实现
```javascript
function speak(text) {
  const utterance = new SpeechSynthesisUtterance(text);
  utterance.lang = 'zh-CN';
  utterance.rate = 0.9;
  speechSynthesis.speak(utterance);
}
```

### 响应式策略
- Mobile-first (max-width: 480px 为基础)
- 平板: 卡片双列
- 桌面: 最大宽度 480px 居中 (保持手机体验)

### 浏览器兼容
- 现代浏览器 (Chrome, Safari, Firefox, Edge)
- iOS Safari 语音 API 支持
- 无需 polyfill
